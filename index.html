Dans le cadre de mon autoformation, j'ai appris à utiliser des api comme par exemple pour des sites de météo ou encore dans un exemple plus concret sur un site regroupant tous les artistes liés à l'API de Spotify. Il est essentiellement développé en HTML CSS et Javascript.

Un peu de 3D... 📷

Je me suis aussi formé à la 3D principalement sur Blender dans un but uniquement personnel. Cependant lier le développement et la 3D avec Three js par exemple reste très intéressant. Voici quelque petit modèle 3D que j'ai fait

Ce site ? ️

Ce site est développé en React avec évidemment une architecture HTML et un style en CSS. J'appele aussi des librairies comme Jquery pour gérer différentes timelines et Three.Js pour la gestion des environnements (light, caméra etc.). Le design est inspiré par celui d'un site qui a reçu un awards créé par Bokoko33. Le modèle 3D est une réplique de mon environnement de travail avec des éléments réels en grande majorité ( drone, comics, manettes, Pc...)

Je suis très satisfait de l'ambiance de ce site mais certains problèmes persiste sur le téléphone. En effet il reste très lent sur Smartphone il va donc falloir que je pense à alléger mon code dans le futur.
